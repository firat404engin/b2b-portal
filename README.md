# B2B Portal Projesi

Bu proje, bayilerin ürün filtreleme, stok/fiyat kontrolü yapabileceği ve sipariş akışını yönetebileceği modern bir B2B (İşletmeden İşletmeye) portal arayüzüdür.

---
